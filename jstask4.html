STRING METHODS

LENGTH OF STRING

The "string.length" is a property which is used to find the length of a given string.The length of empty string is 0.

Parameter: It does not accept any parameter.
Return Values: It returns the length of the given string.

Note: Also length is a property of string object,it must be invoked through a particular instance of string class.

For example,

var x="Pranu"


UPPERCASE

The "string.toUpperCase()" is a property which is used to convert given string to uppercase letters.

For example,

var b="pranathi"

SEARCH

The "string.search()" is a property which is used to search word in a given string .

For example,

var f="Welcome to everyone"

INDEX OF STRING

The "string.indexOf" and "string.indexOf("string",int)" is a method that returns the position of the first occurence of a specified value in a string.This method returns -1 if the value to search for never occurs.

For example,

var y="please locate where 'where' occurs!"

(i). y.indexOf("where");

(ii). y.indexOf("where",18);

LOWERCASE

The "string.tolowerCase()" is a property which is used to convert given string to lowercase letters.

For example,

var c="VINATHI"

SLICE

The "string.slice()" is a property which is used to return the selected elements in a given array or list.

For example,

var g=("Apple,Orange,Banana,Mango,Lemon")

LAST INDEX OF STRING

The "string.lastIndexOf" method returns the position of last occurence of a specified value. Note:The string is searched from end to beginning,but returns the index starting at the beginning,at position 0.

For example,

var a="please locate where 'locate' 'where' occurs!"

(i). a.lastIndexOf("where");

(i). a.lastIndexOf("where",20);

REPLACE

The "string.replace()" is a property which is used to replace one string to another string.

For example,

var d="Welcome to Mits mits"

SUBSTRING

The "string.slice()" is a property which extracts the characters from a string,between two specified indicies,and returns new substring It extracts characters in string between "start" and "end",not including "end" itself.

For example,

var h=("Apple,Orange,Banana,Mango,Lemon")
